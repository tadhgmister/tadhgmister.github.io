bubble sort game

the goal of the game is to sort the bubbles so each vial only contains a single colour. Clicking on a vial will move the top bubbles to another vial with the same colour at the top or an empty vial if no partially full vials are available. Note that pressing the reset button can be undone. With the current implementation you are given a random level every time you refresh the page, it is not known whether the level presented will be solvable. You can change the settings with url query parameters like this: ?ncolors=6&ballspercolor=6&empties=2&emptysize=5. Empty tubes or tubes that can be emptied into a tube with only that colour are highlighted vividly. Tubes that can be emptied into partially full tubes are highlighted less vividly.
